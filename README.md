This project is a community-driven reimagining of the official [Polkadot.com](https://polkadot.com) website. It has been localized for the Italian community while maintaining the high-performance standards and the "Second Age" brand aesthetics of the original site.

---

## üéØ Project Vision

This repository is built following Polkadot's core principles of **open-source** and **sovereignty**.

> **Are you a member of a regional community?**
> You are free (and encouraged!) to fork this repository to create a localized version of Polkadot for your own country (e.g., Polkadot France, Polkadot Spain, Polkadot Japan, etc.). The codebase is lightweight, fast, and easy to translate.

---

## üöÄ Technical Features

* **Minimalist Design**: Fully aligned with the Polkadot "Second Age" brand identity.
* **Extreme Performance**: Built using Vanilla HTML, CSS, and JavaScript. No heavy frameworks, ensuring lightning-fast loading speeds.
* **Interactive Background**: Custom WebGL Shader (Noise Effect) that reacts to mouse movement.
* **Dark/Light Mode**: Native support for theme switching with visual persistence.
* **Mobile Ready**: Fully responsive design with a dedicated mobile hamburger menu.

## üõ† Getting Started

### 1. Fork & Clone
Fork the repository and clone it to your local machine:

`git clone https://github.com/BoredRabbitX/Polkadot_Italia.git`

### 2. Customization
Edit the HTML files to adapt them to your local language or community needs:
- index.html (Home)
- filosofia.html (Philosophy)
- tecnologia.html (Tech)
- piattaforma.html (Platform)
- ecosistema.html (Ecosystem)
- contatti.html (Contact)

### 3. Deploy
Host your site for free in seconds using:
- GitHub Pages
- Vercel
- Netlify
